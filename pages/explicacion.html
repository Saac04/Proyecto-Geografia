1. QGIS

QGIS es un software libre y de código abierto para Sistemas de Información Geográfica (SIG) que permite visualizar, crear, editar, analizar y gestionar datos espaciales y georreferenciados. aporta herramientas para trabajar con mapas y capas de información geográfica, realizar análisis territoriales complejos, procesar datos vectoriales y ráster, crear cartografía de calidad profesional, y conectar con bases de datos espaciales y servicios web de mapas.

2. Anaconda

Anaconda es una distribución de Python que incluye cientos de bibliotecas científicas y herramientas para la gestión de entornos. Fue utilizada en este proyecto para crear un entorno controlado donde todas las dependencias necesarias para el desarrollo funcionaron perfectamente.

3. Leaflet.JS

Leaflet.js es una biblioteca JavaScript de código abierto que permite crear mapas interactivos en páginas web de manera sencilla y ligera. Con ella podemos integrar mapas de diferentes proveedores, añadir marcadores personalizados, dibujar geometrías y mostrar información mediante popups. Es fácil de aprender, funciona en cualquier dispositivo y se ha convertido en una de las herramientas más populares para desarrollar aplicaciones web con mapas interactivos.

Folium

Folium es una herramienta de Python que facilita la generación de mapas web interactivos basados en Leaflet.js. En este proyecto se desarrollaron visualizaciones cartográficas dinámicas que presentan los resultados del análisis espacial de forma clara e intuitiva. Se emplearon iconos diferenciados en cada punto de interés para resaltar la información más relevante del estudio.

  • Representación cartográfica de información meteorológica
  • Visualización multicapa con datos georreferenciados
  • Señalización de ubicaciones significativas

4. Google Earth Engine

Google Earth Engine (GEE) es una plataforma cloud que ofrece capacidades de procesamiento y análisis de datos satelitales y geoespaciales a nivel global. Para este trabajo se empleó en el cálculo de indicadores ambientales como NDWI y MNDWI. Gracias a su potencia de cómputo distribuido, permitió generar y exportar imágenes en formato GeoTIFF con alta calidad y precisión espacial.

Mapa Interactivo con Puntos de Interés

El sitio web integra un mapa interactivo desarrollado con Leaflet.js que facilita la exploración y visualización de ubicaciones georreferenciadas de interés. Los marcadores incluyen datos descriptivos y accesos directos a cuadernos de trabajo o información complementaria. También se han implementado funcionalidades avanzadas como:

  • Cálculo de superficies y mediciones de distancia.
  • Alternancia entre diferentes capas base (vista satélite, cartografía topográfica, etc.).
  • Sistema de clustering para agrupar marcadores y optimizar la experiencia de navegación.

Visor Interactivo con Anaconda

La integración con Anaconda posibilita la ejecución de cuadernos interactivos directamente desde el navegador. Los usuarios pueden explorar, editar y ejecutar código obteniendo resultados inmediatos sin requerir instalaciones previas ni configuraciones complejas. Esta funcionalidad resulta especialmente útil para:

  • Análisis y tratamiento de información GNSS.
  • Creación de cartografía adaptada a necesidades específicas.
  • Desarrollo de scripts para calcular indicadores ambientales.
