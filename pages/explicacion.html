1. QGIS

QGIS es un software libre y de código abierto para Sistemas de Información Geográfica (SIG) que permite visualizar, crear, editar, analizar y gestionar datos espaciales y georreferenciados. aporta herramientas para trabajar con mapas y capas de información geográfica, realizar análisis territoriales complejos, procesar datos vectoriales y ráster, crear cartografía de calidad profesional, y conectar con bases de datos espaciales y servicios web de mapas.

2. Anaconda

Anaconda es una distribución de Python que incluye cientos de bibliotecas científicas y herramientas para la gestión de entornos. Fue utilizada en este proyecto para crear un entorno controlado donde todas las dependencias necesarias para el desarrollo funcionaron perfectamente.

3. Leaflet

Leaflet.js es una biblioteca JavaScript de código abierto que permite crear mapas interactivos en páginas web de manera sencilla y ligera. Con ella podemos integrar mapas de diferentes proveedores, añadir marcadores personalizados, dibujar geometrías y mostrar información mediante popups. Es fácil de aprender, funciona en cualquier dispositivo y se ha convertido en una de las herramientas más populares para desarrollar aplicaciones web con mapas interactivos.

3. Leaflet.JS

Leaflet.js es una biblioteca JavaScript de código abierto que nos permite crear mapas interactivos en páginas web de forma sencilla y ligera. Con ella podemos integrar mapas de diferentes proveedores como OpenStreetMap, añadir marcadores personalizados, dibujar polígonos y líneas, y mostrar información mediante popups cuando el usuario interactúa con el mapa. Es muy fácil de aprender y usar gracias a su documentación clara, funciona perfectamente en móviles y ordenadores, y admite datos en formato GeoJSON. Por todo esto se ha convertido en una de las herramientas más utilizadas para desarrollar aplicaciones web con mapas interactivos.

1. Google Earth Engine

Google Earth Engine (GEE) es una plataforma basada en la nube que permite analizar imágenes satelitales y otros conjuntos de datos geoespaciales a gran escala. En este proyecto, se utilizó para calcular índices medioambientales como el NDWI y MNDWI. Sus capacidades de procesamiento masivo facilitaron la exportación de imágenes GeoTIFF de alta precisión.

2. Folium

Folium es una biblioteca de Python que permite crear mapas interactivos utilizando Leaflet.js. Aquí se integraron mapas personalizados para visualizar resultados de análisis geoespacial en un entorno dinámico y fácil de usar. Cada marcador fue generado con iconos personalizados para destacar los puntos clave del análisis.

  • Visualización de datos climáticos
  • Mapa interactivo con capas superpuestas
  • Marcadores de eventos importantes

Mapa Interactivo con Puntos de Interés

La web incluye un mapa interactivo, construido con Leaflet.js, que permite explorar y visualizar puntos de interés georreferenciados. Cada marcador contiene información relevante y enlaces directos a cuadernillos o datos relacionados. Además, se incorporan herramientas avanzadas, como:

  • Medición de áreas y distancias.
  • Cambio dinámico de vistas (satelital, topográfica, etc.).
  • Agrupación de marcadores mediante clusters para una navegación eficiente.

Visor Interactivo con Anaconda

Gracias a la integración con Anaconda, es posible ejecutar notebooks interactivos directamente en la web. Esto permite a los usuarios explorar y modificar el código, obteniendo resultados en tiempo real sin necesidad de configuración adicional. Este visor es ideal para:

  • Procesamiento de datos GNSS.
  • Generación de mapas personalizados.
  • Ejecutar scripts relacionados con índices medioambientales.

Exportación de Datos GeoTIFF

Una funcionalidad clave de la web es la capacidad de exportar resultados en formato GeoTIFF. Los usuarios pueden seleccionar áreas específicas en el mapa, calcular índices medioambientales como NDWI o MNDWI, y descargar las imágenes procesadas. Esto facilita la integración con herramientas SIG como QGIS o ArcGIS.
